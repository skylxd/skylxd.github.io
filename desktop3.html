GC | Portfolio Introduction About Me My Portfolio - Some of my most recent projects Experience & Skills Pi 5 Touch Server Control Panel  This project involved building a Raspberry Pi 5 system with a 7-inch touchscreen and custom 3D-printed enclosure, running Linux with a custom GUI and automated services, acting as a front-end for server projects and hosting a Minecraft server for up to 12 players. SOHO Home Office & Network  This project involved setting up a SOHO network with three PCs, two Raspberry Pis, 1.5-gigabit Cat 6e cabling, dual ASUS Ai-Mesh access points, and segmented networks from a single gateway to optimize performance for gaming and VR devices. Ender 3 S1 Pro Setup  This project focused on setting up and optimizing my Ender 3, transitioning from Cura to Orca Slicer to significantly improve reliability and reduce print times by up to 160%, while also serving as a strong motivation for me to learn and apply Fusion 360 for 3D design. AI Discord Bot
This utility project has been successful in supporting note-taking and fun tasks for me and my friends, allowing a chat service like Discord to serve as social media and a creative learning tool for taking notes or writing stories; Bridging the gap between Discord and AI. High Performance Workstation  This is my longest-running project and the one that sparked my interest in computer engineering and system design. What began as a personal gaming PC evolved into a foundational learning experience that continues to influence my career path. Hexapod Automated Lookout   HAL was my capstone project for the ECT program at Seneca I graduated from. HAL combined advanced movement, and an entirely 3D printed custom construction with an onboard IoT connected security camera, streamed live to a webpage using dual ESP32 MCU’s.
